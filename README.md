# Архив — персональный трекер медиа

Единый каталог для фильмов, сериалов, аниме, книг, манги и игр — с обложками, статистикой, картой стран и импортом из Steam/Shikimori.

## Запуск

Просто открой `archive.html` в браузере (двойной клик по файлу). Работать нужно именно как с открытым локально файлом, а не через встроенные превью — иначе не будут работать сетевые функции (поиск по базам, импорт).

## Возможности

- Единый каталог 6 типов медиа с разными полями под каждый тип
- Поиск/автозаполнение по открытым базам (iTunes, TVmaze, Jikan, Open Library)
- Импорт из Steam (публичный профиль) и Shikimori
- Импорт/экспорт CSV и JSON
- Статистика: тепловая карта активности, диаграммы, карта мира по странам
- Тёмная/светлая тема, режим "стена постеров"
- Данные хранятся локально в браузере (window.storage)

## Стек

Один HTML-файл, без сборки: vanilla JS + CSS. Карта мира — сжатая SVG из открытого пакета `@svg-maps/world` (MIT).
