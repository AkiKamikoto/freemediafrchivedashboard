Обогащение из API
Дозаполняет описания, режиссеров, авторов, разработчиков и постеры. Фильмы/сериалы и игры не со Steam требуют бесплатных ключей TMDB и RAWG (вкладки «Кино» и «Steam») — без них эти категории пропускаются, а не «не находятся». Steam-игры, аниме, манга и книги обогащаются без ключей.

(.csv или .json)
Экспортируй данные из сервиса (обычно в настройках аккаунта → «Экспорт данных») и загрузи файл сюда.
Letterboxd, IMDb — для них есть отдельная вкладка «Кино» (распознаются автоматически)
Goodreads — отдельная вкладка «Книги»
Кинориум — Настройки → Экспорт коллекции (CSV)
HowLongToBeat — своего экспорта нет, но можно скопировать таблицу в Google Sheets → Сохранить как CSV
Кинопоиск — официального экспорта нет; можно попробовать сторонние выгрузки списков
TMDB — поиск и метаданные
Бесплатно: themoviedb.org/settings/api. С ключом поиск фильмов и сериалов при добавлении записи идёт через TMDB: постеры, описания на русском, режиссёр, актёры, страна, хронометраж.
Letterboxd / IMDb — импорт файлом
Перетащи сюда CSV из Letterboxd или IMDb
(watched / ratings / diary / watchlist из Letterboxd, ratings из IMDb)
Letterboxd — Settings → Data → Export your data (не переименовывай файлы из архива: тип списка определяется по имени файла)
IMDb — Your Ratings → три точки → Export
Постеры и описания
CSV-экспорты не содержат обложек — после импорта дозаполни их из TMDB (нужен ключ выше).
Goodreads
Перетащи сюда CSV-экспорт из Goodreads
(goodreads_library_export.csv)
Экспорт: goodreads.com/review/import → Export Library. Импортируются полки (прочитано / читаю / хочу прочитать), оценки, даты, автор, число страниц и рецензии. Обложки подтягиваются по ISBN из Open Library автоматически.

Метаданные (офлайн-база)
Разработчик, жанр, описание — тянутся из Steam Store API, кешируются в data/steam-games-db.json.
Скачанный файл — в репозиторий как data/steam-games-db.json, чтобы не собирать заново на других устройствах.
Игры других платформ (RAWG)
Бесплатно: rawg.io/apidocs. С ключом поиск при добавлении игры находит PlayStation/Xbox/Switch-игры с обложками, годом и жанрами (вместо магазинного CheapShark).
Прогресс
В панели управления, раздел «Keys».
База игр для поиска (офлайн)
В API RetroAchievements нет поиска по названию — чтобы ретро-игры находились при добавлении записи (например, «sonic»), нужно один раз собрать базу по всем платформам.
Скачанный файл — в репозиторий как data/retroachievements-games-db.json.

MyAnimeList (файл)
Перетащи сюда XML-экспорт из MAL
(animelist_*.xml или mangalist_*.xml)
Экспорт: myanimelist.net/panel.php?go=export. Файл приходит как .xml.gz — сначала распакуй. Обложек в экспорте нет — их можно дозаполнить поиском при редактировании.

Вход

Вход и синхронизация архива между устройствами работают через бесплатный Supabase:
1. Создай проект на supabase.com
2. В SQL Editor выполни скрипт из db/supabase-setup.sql
3. Скопируй Project URL и anon-ключ (Settings → API) в js/config.js
Пока аккаунт не настроен, все данные хранятся локально в этом браузере — как раньше.
После входа архив автоматически синхронизируется между всеми устройствами, где ты вошёл.
